# Diagrams

This directory will store diagrams for the AWS Logging, Monitoring & Detection project.

## Planned Diagrams

Potential diagrams may include:

- AWS logging architecture diagram
- CloudTrail investigation workflow
- GuardDuty finding triage workflow
- EventBridge alert routing diagram
- AWS monitoring flow diagram

## Diagram Requirements

Diagrams should clearly show:

- Log source
- Event destination
- Review location
- Alerting path
- Investigation workflow
- Optional response step

## Security Notes

Diagrams should not expose sensitive information.

Do not include:

- AWS account ID
- Access keys
- Secret keys
- Personal email addresses
- Billing information
- Unredacted public IPs unless intentionally documented
- Sensitive resource identifiers
