# Screenshots

This directory will store redacted screenshots for the AWS Logging, Monitoring & Detection project.

## Screenshot Rules

Before uploading screenshots, verify that they do not expose sensitive information.

Do not commit screenshots showing:

- AWS account ID
- Access keys
- Secret access keys
- Billing details
- Personal email addresses
- Unredacted ARNs if sensitive
- Unredacted public IP addresses unless intentionally documented
- Sensitive CloudTrail event fields
- Sensitive GuardDuty finding details
- Private information from the AWS console

## Planned Screenshot Evidence

Potential screenshots may include:

- CloudTrail event history
- Example CloudTrail event
- CloudWatch log group
- CloudWatch alarm
- GuardDuty dashboard
- GuardDuty sample finding
- EventBridge rule
- Alerting workflow test

## Redaction Requirement

All screenshots must be reviewed and redacted before being committed.
